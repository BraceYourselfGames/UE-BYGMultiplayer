# BYG Multiplayer

## Installation

### Source

1. Download the zip or clone the repository to
   `ProjectName/Plugins/BYGMultiplayer`.
2. Add `BYGMultiplayer` to `PrivateDependencyModuleNames` inside `ProjectName.Build.cs`.

## Unreal Version Support

* Unreal Engine 4.27+
